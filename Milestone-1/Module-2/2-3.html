Singleton Design Pattern:
Definition: The Singleton design pattern ensures that a class has only one instance and provides a global point of access to that instance.

To implement a Singleton:
Make the constructor private, so that no other class can instantiate it directly. Provide a static method that acts as a constructor and returns the same instance every time it's called.
